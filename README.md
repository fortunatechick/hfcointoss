# hfcointoss
A fan-made 3D coin flip web app dedicated to 8bit HeadFlicker and his streams. Vibe-coded with Google's Antigravity!

> Note: This website was entirely vibe-coded with Google's Antigravity and not manually coded!

## Features
- **Realistic 3D Coin Flip Animation**: Physics-based coin flip with dynamic shadows, spin, and wobble effects.
- **Single & Best of 3 Modes**: Predict individual flips or play a Best of 3 series.
- **Live Stats & History**: Tracks total flips, heads/tails probability, and streaks.
- **Sound & Visual Effects**: Custom sound effects, screen flashes, and confetti for a highly engaging experience.
- **Stream Integration**: Built-in Kick chat embed for seamless interaction during live streams.

## Setup
No special installation is required. Simply open `index.html` in your web browser to run the application.

## Technologies Used
- HTML5
- CSS3 (Custom Variables, 3D Transforms, Animations)
- JavaScript (Web Audio API, Physics Simulation, DOM Manipulation)

## Usage
- Click the **Flip Coin** button or press `Space` / `F` to flip.
- Press `H` or `T` to quickly predict the outcome.
- Use the side panel buttons to clear stats or toggle sound effects.
